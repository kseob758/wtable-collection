# wtable-collection
- 수집 페이지 : 우리의식탁 [https://wtable.co.kr/recipes](https://wtable.co.kr/recipes)
- 수집 데이터 : 요리이름, 카테고리, 난이도, 소요시간, 상세페이지 링크, 이미지 링크, 재료, 요리방법, 요리설명
- 수집 방법 : requests, BeautifulSoup 사용
